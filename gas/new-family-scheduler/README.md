# 새가족 심방 스케줄러

Google Sheets를 DB처럼 사용하는 Google Apps Script 웹앱입니다.

## 포함 파일

- `Code.gs`: 시트 생성, 명단 조회, 새가족 등록, 심방 완료 기록, 샘플 데이터 입력
- `Index.html`: 메인 관리자 화면
- `Styles.html`: 화면 스타일
- `JavaScript.html`: 클라이언트 이벤트 처리
- `appsscript.json`: GAS 매니페스트

## 시트 구조

앱이 처음 실행되면 아래 시트를 자동 생성합니다.

1. `새가족명단`
   - `이름`
   - `등록일`
   - `담당사역자`
   - `마지막심방일`
   - `심방주기(일)`
   - `연락처`
2. `심방기록`
   - `이름`
   - `담당사역자`
   - `심방일`
   - `메모`
   - `기록일`

## 사용 방법

1. 구글 스프레드시트를 하나 새로 만듭니다.
2. `확장 프로그램 > Apps Script`로 들어갑니다.
3. 이 폴더의 파일 내용을 각각 같은 이름으로 생성해 붙여 넣습니다.
4. 저장 후 `배포 > 새 배포 > 웹 앱`으로 배포합니다.
5. 최초 실행 시 권한 승인을 진행합니다.
6. 앱 화면에서 `샘플 데이터 넣기`를 누르거나 직접 새가족을 등록합니다.

## 동작 방식

- `마지막심방일 + 심방주기(일)`로 `다음예정일`을 계산합니다.
- 예정일이 지난 경우 `지연`, 오늘이면 `오늘`, 7일 이내면 `임박`으로 표시합니다.
- `심방 완료` 버튼을 누르면:
  - `새가족명단`의 `마지막심방일`이 갱신되고
  - `심방기록` 시트에 이력이 추가됩니다.

## 기본 가정

- 한 사람당 최근 심방일 1개만 `새가족명단`에 유지합니다.
- 자세한 이력은 `심방기록` 시트에 누적 저장합니다.
- 날짜 형식은 `yyyy-MM-dd` 기준입니다.
