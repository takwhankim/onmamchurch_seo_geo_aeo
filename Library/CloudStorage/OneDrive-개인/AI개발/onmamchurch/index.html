1초 설치

북마클릿으로 바로 배포

이 서비스는 앱 설치형이 아니라 북마클릿으로 제공됩니다. 교회 홈페이지를 열고 북마크를 누르는 것만으로 분석 패널이 실행되는 사용 흐름을 기준으로 랜딩을 구성했습니다.

북마클릿

아래 버튼을 북마크 바로 드래그하세요

클릭하지 말고 드래그해서 북마크 바에 놓으세요. 이제 외부 파일을 다시 불러오지 않고, 북마클릿 안에 실행 코드가 직접 들어갑니다.

버튼 드래그

3

진단하려는 교회 홈페이지를 연 뒤 북마클릿을 클릭하면 분석 패널이 뜹니다.

FAQ

배포 방식이 북마클릿이기 때문에, 설치와 실행 방식에 대한 설명을 FAQ로 같이 풀었습니다.

교회 실무자나 제작사가 별도 프로그램 설치 없이 지금 보고 있는 홈페이지를 바로 진단할 수 있기 때문입니다. 배포와 업데이트도 스크립트 교체로 간단하게 관리할 수 있습니다.

교회 메인 홈페이지, 예배안내 페이지, 설교 페이지, 새가족 등록 페이지 등 실제 진단이 필요한 어떤 페이지에서도 실행할 수 있습니다.

우측 슬라이드 패널이나 모달 형태로 점수, 체크리스트, 우선 수정 항목, 캡처/복사 기능을 보여주는 방식이 가장 자연스럽습니다.

물론입니다. 예배시간, 헌금안내, 온라인예배, 교회학교, 목회자 소개, 카카오톡 문의, 지도 연결처럼 실제 운영 흐름에 맞춘 규칙을 계속 추가할 수 있습니다.
